About The Odin Project <-- Using the "target" and "rel" I can target the link to open another page by using (target="_blank"). In the "rel" value I used "noopener noreferrer", the first part "noopener" links that blank page to the link we want. The second part "noreferrer" creates privacy and security by not showing the person who opened the link how you got there so it is truly a fresh page.

Open Example <-- This is and example if you use the "noopener" do not use the "noreferrer" value.

Visit Example <-- This example has no "noopener" value but has a "noreferrer" value in the "rel".

In this example: target=”_blank”: opens the link in a new tab. rel=”noreferrer”: ensures the new page cannot see the referring page’s address (privacy) and prevents it from accessing the original page (security).

By using rel=”noreferrer”, you automatically get the benefits of both privacy and security.
